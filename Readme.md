# Hedwig-Gallery


This is a Hugo-based photoblog that uses the [Hugo Shortcode Gallery](https://github.com/mfg92/hugo-shortcode-gallery) and the [Ananke theme](https://github.com/theNewDynamic/gohugo-theme-ananke). The photoblog showcases various images with descriptions, and it is deployed using GitHub Pages.
